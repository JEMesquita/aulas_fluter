# aulas_fluter
# Repositorio dos aplicativos das Aulas de desenvolvimento para dispositivos moveis

Este repositório foi criado para organizar e centralizar os projetos desenvolvidos durante as aulas de **Desenvolvimento para Dispositivos Móveis**.

---

## 🚀 Instruções de Uso

Para manter a organização e o histórico do aprendizado, siga os passos abaixo:

1. **Clonar o Repositório:** Utilize o comando `git clone` para baixar o projeto.
2. **Incluir Arquivos:** Adicione suas atividades nas pastas corretas.
3. **Comentar Commits:** Descreva brevemente o que foi feito em cada atualização.

---

## 🛠️ Principais Comandos Git

Abaixo estão os comandos fundamentais que utilizaremos no dia a dia:

### Configuração Inicial
* `git config --global user.name "Seu Nome"`: Configura seu nome de usuário.
* `git config --global user.email "seu@email.com"`: Configura seu e-mail do GitHub.

### Fluxo de Trabalho
* **`git clone <url>`**: Faz o download do repositório para sua máquina.
* **`git status`**: Lista os arquivos modificados e o que ainda não foi salvo.
* **`git add .`**: Adiciona todos os arquivos modificados para a fila de envio (Stage).
* **`git commit -m "Mensagem descritiva"`**: Salva as alterações localmente com um comentário.
* **`git push origin <nome-da-branch>`**: Envia seus commits locais para o GitHub.
* **`git pull`**: Atualiza seu repositório local com as últimas mudanças do servidor.

### Ramificações (Branches)
* **`git branch`**: Lista as branches existentes.
* **`git checkout -b <nome-da-branch>`**: Cria uma nova ramificação e entra nela.
* **`git merge <nome-da-branch>`**: Une as alterações de outra branch à sua branch atual.

---
*Dica: Sempre execute um `git pull` antes de começar a trabalhar para evitar conflitos de código!* 📱💻
